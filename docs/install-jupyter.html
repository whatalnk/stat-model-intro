Jupyter インストール

Python

Windows

  • Miniconda をインストールする
    • Python 3.5

Mac

  • Homebrew をインストールする
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  • Homebrew から pyenv をインストールする
brew update
brew install pyenv
  • pyenv からMiniconda をインストールする
pyenv install miniconda3-latest

Jupyter

conda install jupyter
